محاكي متقدم وواقعي لموجه الأوامر (Command Prompt) في ويندوز، يعمل بالكامل داخل المتصفح دون الحاجة لتثبيت أي برامج.

يوفر تجربة تعليمية ممتعة وتفاعلية لتعلم أوامر CMD الشائعة والمهمة بطريقة آمنة وبسيطة.

## الميزات الرئيسية

- **أكثر من 40 أمر مدعوم بشكل كامل** (dir, cd, ping, ipconfig, tasklist, systeminfo, shutdown, tree, وغيرها).
- **محاكاة واقعية** لإخراج الأوامر تمامًا كما في CMD الحقيقي.
- **سجل الأوامر** مع دعم التنقل باستخدام الأسهم ↑ ↓.
- **تغيير الثيم** (أخضر كلاسيكي، أبيض، أزرق، كهرماني).
- **تغيير حجم الخط** لتحسين القراءة.
- **بحث ذكي** عن الأوامر مع تصنيفها حسب الفئة.
- **واجهة جميلة وسريعة الاستجابة** تدعم الهواتف والأجهزة اللوحية.
- **لا يتطلب خادم أو تثبيت** – مجرد ملف HTML واحد!

## الأوامر المدعومة (مقتطف)

| الفئة                  | أمثلة على الأوامر                              |
|-------------------------|------------------------------------------------|
| أساسيات النظام         | `help`, `cls`, `dir`, `cd`, `md`, `rd`, `echo`, `title`, `ver` |
| معلومات النظام         | `whoami`, `systeminfo`, `hostname`, `date`, `time` |
| الشبكة                  | `ipconfig`, `ping`, `tracert`, `netstat`        |
| إدارة العمليات         | `tasklist`, `taskkill`                          |
| صيانة النظام           | `sfc /scannow`, `chkdsk`, `shutdown`            |
| أخرى                    | `tree`, `color`, `exit`                         |

اكتب `help` داخل المحاكي للحصول على القائمة الكاملة.
